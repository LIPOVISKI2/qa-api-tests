# 🔍 QA API Portfolio - Felipe Oliveira Rodrigues

## 📌 Sobre
Projeto prático de testes de API utilizando Postman, com foco em validação de requisições HTTP, análise de respostas e identificação de falhas.

---

## 🧪 Testes realizados

### ✔ GET /users
- Validação de status 200
- Validação de estrutura da resposta
- Verificação de retorno de lista de usuários

### ✔ POST /users
- Criação de usuário com dados válidos
- Validação de status 201 (Created)
- Verificação de retorno de ID na resposta

---

## 🛠️ Ferramentas utilizadas
- Postman
- Testes manuais
- GitHub

---

## 🐞 Bug identificado

### API aceita dados inválidos sem validação

**Descrição:**
A API permite envio de dados inválidos sem retornar erro.

**Impacto:**
Pode comprometer integridade dos dados no sistema.
